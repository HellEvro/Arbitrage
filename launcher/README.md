# Arbitrage Launcher

`launcher/arbitrage_manager.py` — графический менеджер на базе Tkinter для обслуживания проекта Arbitrage Bot.  
Он предоставляет удобный интерфейс для всех повседневных операций, включая запуск бота и установку зависимостей.

Для сценариев автоматизации или работы через терминал доступен текстовый лончер `launcher/arbitrage_launcher.py`.

## Возможности

- Создание или переустановка виртуального окружения `.venv`
- Установка зависимостей из `requirements.txt` и `requirements-dev.txt`
- Установка проекта в режиме `pip install -e ".[dev]"`
- Запуск `main.py` внутри виртуального окружения
- Просмотр статуса окружения (версии Python, наличие конфигурации и т.д.)
- Быстрый доступ к конфигурации (`config/config.yaml`) и журналам (`logs/`)
- Просмотр хвоста конкретного лог-файла
- Запуск тестов (`pytest`)

## Запуск

После выполнения установочного скрипта (`installer/install_<os>.{ps1|sh}`) используйте соответствующий лончер:

- Windows: `launcher\start_launcher.cmd`
- Linux/macOS: `./launcher/start_launcher.sh`

Оба скрипта автоматически запускают графический интерфейс и используют `.venv`, если оно уже создано.
Также можно запускать нужный инструмент напрямую:

```bash
# Запустить графический менеджер
python launcher/arbitrage_manager.py

# Запустить текстовый лончер без аргументов (интерактивное меню)
python launcher/arbitrage_launcher.py
```

Без аргументов откроется интерактивное меню. Для сценариев автоматизации доступны команды:

```bash
# Создать окружение и установить зависимости
python launcher/arbitrage_launcher.py setup

# Запустить бота с дополнительными аргументами
python launcher/arbitrage_launcher.py run -- --min-profit 1.5

# Показать статус окружения
python launcher/arbitrage_launcher.py status
```

## Требования

- Python 3.11+
- Доступ к интернету для установки зависимостей и запросов к биржам

`tkinter` требуется для графического менеджера.

Текстовый лончер `arbitrage_launcher.py` работает в терминале и поддерживает Windows, Linux и macOS.
